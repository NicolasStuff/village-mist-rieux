# ImageGen Prompts

Les assets finaux utilises par l'application sont les fichiers `.webp` dans `public/assets`.
Les planches sources ImageGen restent dans `~/.codex/generated_images/019f093a-69e9-7b30-9d1d-1b4c58d8a100`.

Direction commune:

```text
Original card-front artwork for a family social deduction web game named Le Village Mysterieux. Classic hidden-role tabletop card aesthetic, rustic medieval French village, ink drawing with watercolor/gouache wash, parchment texture, thick expressive outlines, ornate but simple old-card border, moody warm lantern light, kid-friendly mystery, no horror, no gore, no readable text, no logos, not copying any existing card game.
```

Planches generees:

- Roles: planche 4 x 3 avec Notaire, Archiviste, Garde champetre, Faussaire, Maire, Serrurier, Commere, Guerisseuse, Cartographe, Horloger, Messager, Crieur public.
- Affaires A: planche 3 x 3 avec cave/lettre, vieux puits, moulin, chapelle, cloche, registre, pont, marche, boulangerie.
- Affaires B: planche 3 x 2 avec manoir, etang, bal, gare fermee, promesse du maire, inconnu de la gare.
- Dos de carte: carte verticale 3:4 avec village au clair de lune, enveloppe scellee, cle, lanterne et brume.

Les planches ont ete decoupees localement en 720 x 960 `.webp`, puis integrees aux chemins references par les catalogues de roles et secrets.
